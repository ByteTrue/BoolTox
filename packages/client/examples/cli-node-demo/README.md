# 文件管理器（CLI）- Node.js 版

基于 Inquirer 的交互式 TUI 文件管理工具。

## 功能

- ✅ 交互式菜单（方向键选择）
- ✅ 列出目录内容
- ✅ 搜索文件
- ✅ 查看文件信息
- ✅ 创建目录
- ✅ 目录导航

## 独立运行

```bash
# 安装依赖
npm install

# 运行（进入交互模式）
node cli.js
```

**交互示例**:
```
============================================================
  📁 文件管理器 - BoolTox CLI 工具
============================================================

? 当前目录: /Users/... (Use arrow keys)
❯ 📂 列出文件
  🔍 搜索文件
  📄 查看文件信息
  📁 创建目录
  🔙 返回上级目录
  📍 切换目录
  ──────────
  ❌ 退出
```

**使用方式**:
1. 用 **方向键 ↑↓** 选择操作
2. 按 **Enter** 确认
3. 根据提示输入或选择
4. 自动返回主菜单

## 特点

- ✅ **TUI 菜单**: 方向键选择，Enter 确认
- ✅ **交互式操作**: 无需记忆命令
- ✅ **彩色输出**: 目录/文件区分
- ✅ **零改造**: 标准 inquirer 工具
