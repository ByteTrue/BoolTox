# BoolTox v0.0.3 更新说明

## 🎉 重磅更新

### 1. 全新的公告系统

- 支持 Markdown 格式的公告内容
- 支持图片、代码块、列表等富文本展示
- 公告内容现在可以独立文件管理，更加清晰

### 2. 目录结构优化

- 资源文件现在按类型分类存储
- `releases/` 存放版本更新说明
- `news/` 存放日常动态

## 🐛 问题修复

- 修复了公告只能显示纯文本的问题
- 优化了网络请求逻辑，支持并发加载公告内容

## 🔜 后续计划

- 支持公告的本地缓存
- 支持未读公告红点提醒
